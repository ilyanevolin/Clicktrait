# Clicktrait (2016)

Clicktrait is a Conversion Rate Optimization SaaS product.
It uses the A/B and MultiVariate testing concepts and basic statistical knowledge.

## Features

* A/B testing
* MultiVariate testing
* Heatmaps
* Basic analytics
